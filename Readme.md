Projekt Inżynieria Internetu
========

Zależności:
--------
- python3.2
- biblioteka ConfigParser dla pythona3.2
- gnuplot dla przeglądania wygenerowanych wykresów

Sposób użycia:
-----------
python3 main.py
gnuplot -p print.plt

Konfiguracja:
----------
Wszystkie aktualne zmienne można ustawić w pliku config.ini
